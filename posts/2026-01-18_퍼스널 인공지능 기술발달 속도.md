4. **'지능'** 의 소유권이

**'기관'** 에서 **'개인'** 으로

도약하는 시대가 **지금** 임

​

5. 비트코인 금융혁신 어쩌고,

초전도체 미래 기술 어쩌고,

​

그런 것은 그냥 뉴스 이야기지만

​

**'개인'** 이 **'인공지능'** 을 소유하고,

자신의 일상을 혁신한다 이거는요

​

그걸 실제로 누려본 사람과

아직 아닌 사람 밖에 없어요

​

6. 기다리면 풀리지 않을까요?

​

아뇨, **'무조건 규제'** 걸립니다

​

개인이 아니라

기업 레벨에서도

​

특정 집단이 소유하면

**안 될 기술** 입니다

​

제가 장담하는데, 대중 레벨로 풀려도

**온갖 검열, 출력 제한** 다 걸어둘거구요,

​

**\* 현재 오픈소스 인공지능**

**보안 수준은 없는 수준 입니다**

**​**

**일단 있으면, 글자 그대로**

**자기 마음대로 할 수 있어요**

​

극단적으로는 다량의 데이터 관리,

수집, 보유를 **엄격하게** 규제할 겁니다

​

7. 성능 차이가 실감이 잘 안 나는데요?

​

1) 무료 → 유료

2) 유료 → 도메인

3) 도메인 → API

4) API → 보조모델

​

여기까지 써보시면,

이걸 모를 수가 없어요

​

이미지 모델 수치 기준으로 **200배 이상**,

API 모델 수치 기준으로 **16배 차이** 납니다

​

이게 **'가내수공업'** 로컬 수준이구요,

​

전문가, 엔지니어 손에 들어가면?

그 단위는 헤아릴 수도 없어요
