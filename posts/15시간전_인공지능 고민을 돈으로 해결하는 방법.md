# 인공지능 고민을 돈으로 해결하는 방법
**Date:** 15시간 전
**Category:** 다이어리
**Original URL:** https://blog.naver.com/xpfkwh56/224189053537
---

**1. 필요 예산**

**​**

약 300달러 정도 추정

​

**\* 각자 사용량에 따라**

**더 적을 수도, 많을 수도**

​

**2. 장점**

​

아주 빠르게 **맛보기** 가능

단기에 활용성 **부스팅** 가능

​

저점을 빠르게 충족할 수 있음

​

**3. 단점**

​

월 비용을 계속 지불해야 됨

현실적인 측면에서 반영구적

​

**고점에 닿는 속도가 느려질 수 있음**

​

수학 문제 푸는 것으로 비유하면,

​

2점, 3점을 엄청 빨리 풀 수 있는

방식으로 접근하는 것입니다

​

대신 4점 문제를 풀거나, 시험지 밖에

있는 문제를 풀 수 있는 역량을 얻기에는

어려울 가능성이 제 생각에는 더 높습니다

​

영어로 비유하면, 시험지 영어는 잘 하지만

영어를 직접 생활에 쓰긴 늦어질 수도 있음

​

아주 빠르게 70점 까지 찍는 것이기 때문에,

남은 30점만 잘 기억하면 **용도에 따라** 좋음

​

**4. 접근 철학**

​

1) 잘 모르겠으면 대기업이 답이다

2) 쓸 물건은 **산업용으로** 구매해라

​

**\* 음식은 식자재 마트에서 구매하고,**

**업자들이 사용하는 것 위주로 선택한다**

**​**

**아기 용품은 맘카페 베스트 상품 말고**

**조리원, 병원에서 쓰는 것을 쓰도록 한다**

**​**

**아기 교육 교구는 문화 센터나,**

**유명 어린이집에서 쓰는 것을 쓴다**

​

인공지능 시장에 1번은 잘 안 통함

근데 2번은, 좀 더 유효한 부분 있음

​

소비자 시장에서 지피티 점유율이 50%,

제미나이가 30%, 그록이 10% 정도임

​

하지만 기업용 어시스턴트 시장에서는

클로드가 30%가 넘고 일부 서비스 경우,

거의 90%가 넘는 독과점 점유율을 가짐

​

5. 여러 문제를 돈으로 해결하는 것이므로

복잡한 고민 다 치우고, 전부 결제로 갑니다

​

1) 클로드 회원가입

2) **클로드 프로** 결제

​

**\* 5X, 20X 아닙니다**

**​**

**그냥 프로 결제만 하시구,**

**쓰다가 부족하면 늘리세요**

**​**

**돈 쓸 생각하면 쓸 곳이 널렸으니**

**가능한 필요한 곳에 필요한 만큼만**

**써야지 나중에 발목을 덜 잡힙니다**

**​**

3) **기능**에 들어간 다음에,

​

아티팩트, 도메인 접근 권한 등

얘가 해줄 법한 것 전부 권한 허용

​

커넥터 들어가서,

있으면 좋을 것 같은 것

전부 허용하고 연결

​

**\* 커넥터 = 보조 모델**

**​**

나중에 정확히 그게 뭔지 다 알면

그 무렵에 이제 끄고, 키면서 쓰세요

**​**

**\* 기능 다 켜고 있으면 돈 펑펑 쓰는 것임**

**​**

**운전할 때, 신호대기 걸리면 기어 조정할래?**

**브레이크만 밟고 기다릴래 정도 차이 지만요**

​

3) **웹 클로드** 에게 이렇게 질문

​

**클로드 코드를 데스크탑**

**터미널에서 사용하고 싶어,**

**어떻게 해야 되는지 알려줘**

**​**

설치 방법을 알려줄 것인데,

파이썬 설치부터 다 알려줄 것임

​

혹시 틀리게 알려줘도 마음을 비웁니다

​

왜냐? 얘 없었으면 나 혼자는 못 했을 일이니

겸허하게 그냥 다시 지우고, 새로 깔면 됩니다

​

**4) 클로드 코드 설치 후, 연결**

**​**

**옵시디언** 설치

​

5) 그럼 **2개의 AI** 를 쓰게 될 것임

​

6. 웹에 있는 클로드는 **'지시자'** 고,

데탑에 있는 클로드 코드는 **'실무자'** 임

​

클로드 코드한테 이렇게 요청합니다

​

**바탕화면에 테스트 라고 적힌**

**텍스트 파일 1개 만들어줘**

​

그럼 만들어줄 겁니다

​

**다음은 바탕화면에 있는**

**방금 만든 파일 지워줘**

**​**

하면 지워줄 겁니다

​

클로드 코드는 이제 데탑에 있는

프로그램에 대한 **권한**이 있습니다

​

내 컴퓨터에 있는 불필요한 프로그램들 찾아줘,

용도별로 즐겨찾기, 북마크들 다 새로 정리해줘

​

카톡 켜달라고 하면 켜주고,

유튜브 틀어달라고 하면 틀어주고,

​

프로그램 사용법 아냐고 물어보고,

안다고 하면 시키면 되고, 모른다 하면

본인이 가르쳐 주면서 쓰면 됩니다

​

**나도 모르는데, 어떻게 가르쳐요?**

​

웹 클로드한테 해당 지식이 있는

문서, 또는 사이트를 준 다음에

​

공부해달라고 하고, 어떻게 지시해야

클로드 코드가 알아듣나 모르겠다고

알려달라고 하면 걔가 알려줄 겁니다

​

**\* ~를 시켜야 되는데, 템플릿 알려줘**

**​**

중간에 영어 나오면 **제미나이** 접속

​

**빠른 답변** 설정을 켜신 다음에,

영어 직역해달라고 하시면 됩니다

​

돈 많으면 앞으로 나랑 하는 대화는

다 한글로 해달라고 해도 무관합니다

​

**\* 한글이 최소 2-3배, 많으면 10배 비쌈**

**​**

돈은 없고, 영어는 할 줄 모른다

​

자본 X, 기술 X 일 때

할 수 있는 것은

노력이 **유일** 합니다

​

대화를 할 때마다, 파파고든 딥플이든

지피티든, 제미나이든 번역기에 넣고

​

왔다갔다 복붙으로 쓰시면 됩니다

​

모양은 조금 짜칠 수 있지만,

저는 이게 어지간한 학원보다야

​

**훨씬 더 영어를 빨리 배우는**

**방식이라고 생각하는 편입니다**

**​**

**\* 제 경우, 대부분 다 이렇게 배웁니다**

**​**

7. 웹 클로드, 클로드 코드랑 대화하다

컨텍스트 한도 걸릴 즈음 무렵이 되시면,

**​**

**\* 한글이 더 많은 용량을 점유한다**

**→ 더 빨리 컨텍스트 하드캡에 걸린다**

**​**

**→ LG, 삼성, 네이버에서 죽어라 만들어도**

**이 한계를 못 깨면 인공지능 국산화 기술이**

**근본적인 문제를 결국 해결할 수 없는 것임**

​

컨텍스트 윈도우를 수동으로 저장해서

연동하고 싶다고 요청하고, 해당 내용을

맥다운으로 만들어 달라고 요청하세요

​

**\* 맥다운이 뭔지 모른다 → 물어보면 됨**

​

그리고 클로드 코드한테, 내가 만든

맥다운을 옵시디언에 **연동** 시켜달라

​

라고 하시면, 옵시디언에 일종의 보고서

형태로 정리해서 알아서 올려줄 겁니다

​

왜 **하필** 옵시디언 인가요?

​

더 괜찮은 것 있으시면

그거 하셔도 됩니다

​

8. 이 무렵 즈음에, 로컬 배울 필요가 없네?

​

시발 이거 뭐야, 그냥 다 할 수 있네? 라는

생각을 갖게 되실 가능성이 **'상당히'** 큰데요

​

보이는 것에 속지 마시고, 유료 토큰이나

유료 API **'덜'** 쓰는 방향으로 가능한 바꾸고,

​

수직 계열화 해서 본인 스스로

자립하는 형태로 가게 되실 겁니다

**​**

**\* 월 1-2천 달러 정도 API 결제로 쓰겠다,**

**그러면 안 되는 것 빼고는 다 할 수 있습니다**

​

그럼 나중에는 지피티든, 제미나이든,

그록이든, 클로드든, 그 외 무엇이든간에

​

외부 서비스 없이 독자적인 생태계 안에서

인터넷 연결 없이도 굴러가는 AI 가 챙겨짐

​

**9. 여기서 로컬로 어떻게 체제 전환?**

​

웹 클로드 → 지시자

클로드 코드 → 실무자 에서,

​

클로드 코드 → 상급 실무자

로컬 AI 모델 → 하급 실무자

​

2개로 분류해서, 클로드 코드한테

로컬 AI 모델에게 지시할 내용을

​

정리해서 나한테 보고 하고,

지시하라고 쪼개서 주면 됩니다

​

저는 인공지능 네트워크를

부품이 연결된 하나의 공장, 또는

기업이라고 연결하고 접근합니다

​

위와 같은 논리는 기본적인

접근 철학에도 적용되는데요,

​

광고비 많이 내서, 고객들 들어온다고

오늘 내일 행복에 젖으면 어찌될까요?

​

고객이 오는 것은 오는 것이고,

전체 유입 시장에서 내 마케팅 외주로

​

들어오는 비율이 **'증가'** 해야

아웃소싱 없이도 **'자립'** 되는 것임

​

과외는 성적 올리려고 하는 것이 아닙니다

과외 선생님이 **없어도 되게 하려고** 받는 것
