**1) 손으로 옷 만들기 vs 재봉틀 쓰기**

​

전자의 생산성은 후자를 따라갈 수 없음

여기서 재봉틀이 **워크 스테이션** 임

​

5090 글카는 현존하는

소비자용 **'최고'** GPU 인데,

5090 있으면 옷 3벌 5벌 만들 수 있고

특정한 옷은 5090 이 없으면,

**'아예'** 걍 시작도 할 수가 없음

​

**왜 하필 5090 인데요?**

​

컴퓨터에 cpu 랑 gpu 가 있는데,

​

cpu = 고학력 박사

gpu = 저학력 단순 근로자

​

cpu 가 좋다 = 고급 연산이 빠르다

gpu 가 좋다 = 단순 연산이 빠르다

​

좋은 cpu = 설포카 박사 1명 쓰기

좋은 gpu = 노예 100,000명 쓰기

​

인공지능 작업은 서울 전역에 있는

전봇대에 전단지를 붙이는 작업 임

​

이걸 박사가 하나, 초졸이 하나

어차피 차이가 별로 없는데

​

물리적으로 박사 1명이 아무리

열심히 한다고 해도 몇 개 붙이겠음?

​

그냥 노예 100,000명이 더 잘 함

​

그래서 좋은 gpu 가 있을수록,

더 빨리, 더 정교하게 작업할 수 있음

**​**

가장 작은 정수의 비를 어떻게 구할까?

​

라는 질문을 가진 사람은

최대 공약수를 배우지 않아도,

결국 알 수밖에 없어짐

​

해상도는 가로와 세로의 픽셀 수를 의미

가로와 세로의 길이를 최대 공약수로 나누면

특정 이미지의 비율을 구할 수 있음

​

FHD 화질 이미지는 해상도가 1920\*1080

4K 화질 이미지는 해상도가 3840\*2160

​

1920 과 1080 의 최대 공약수는 120

1920 / 120 = 16

1080 / 120 = 9

​

그래서 1920 \*1080 은 16 : 9

마찬가지로 4K 도 16 : 9

​

업스케일링 기술은 낮은 해상도의 이미지를

더 높은 해상도의 이미지로 키우는 작업을 말함

​

1080p(FHD) 영상의 이미지를

2160p(4K) 영상의 이미지로 키우려면?

​

가로와 세로가 정확히 2배 이기 때문에,

전체 면적은 2^2, 전체 면적의 4배가 되고,

​

정수배로 증가시킬 수 있으니까

화면이 찌그러지는 것 없이 확실히 가능

​

그러나, 720p(HD) 영상을 1440p(QHD)로

보낼 때는 정수배로 떨어지지만,

​

1080p를 1440p로 보낼 때는 1.33배 나옴

이 때, 픽셀을 1.3xxx로 쪼갤 수 없으므로

​
