# 피자 사이즈 고르기
**Date:** 2026. 1. 15. 15:45
**Category:** 다이어리
**Original URL:** https://blog.naver.com/xpfkwh56/224147689730
---

1. 원넓이 = 파이 \* r^2

​

지름 1cm 가 늘어나서

늘어난 피자 면적이 궁금하면

​

늘어난 피자 면적에서,

원래 피자 면적 빼면 됨

​

S1 = 원래 피자 넓이

S2 = 지름 증가한 피자 넓이

​

S2-S1 = 늘어난 피자 넓이

​

만약 이걸 알면, 내가 레귤러랑

라지 골랐을 때 얼마나 실제로,

제대로 값을 내고 먹나 알 수 있음

​

2. 그렇다면 **'진짜'** 그런가?

​

내가 직접 지름 20cm 피자랑,

40cm 피자로 해보면 된다

​

**S1 = 파이 \* 10^2 = 100 파이**

**S2 = 파이 \* 20^2 = 400 파이**

​

파이값을 편의상 3.14 로 잡으면,

​

**\* 소수점을 더 반영할수록 더 정밀,**

**대신 복잡함과 트레이드 오프 관계임**

**​**

S1 = 3.14\*100 = 314

S2 = 3.14\*400 = 1256

​

몇 개 더 계산을 해보면,

​

지름이 2배 늘어날 때마다

면적이 4배 늘어난다는 것을

​

결국 일반화 하는 것이 가능해지는데,

이게 배우지 않고 알 수 있는 사례임

​

**3. 그럼 여기서 끝?**

​

위 계산은 단순히 **'크기'** 만 따짐

​

1) 업체에서 제공하는

길이 자료가 정확한지,

​

**\* 이건 사람이 하는 일이니,**

**당연하게도 오차가 있을 것임**

**​**

2) 크기가 늘어날 때 마다,

재료는 얼마나 늘리는지

​

**\* 식당에선 사이즈를 늘린다고,**

**재료를 선형적으로 늘리지 않음**

**​**

**→ 알바를 했던 사람이라면 앎**

**​**

3) 면적은 정확히 반영되는지,

​

**\* 밀가루가 부풀면서, 입자마다**

**여백이 생기는데 혹시라도 그게**

**더 커지면 커질수록 늘진 않는지**

**물질이 직관적 속성과 일치하는지**

**​**

4) 토핑의 촘촘함도 반영되는지,

​

이런 것을 더 따지고 따질수록

결국 더 **정확한** 것을 알 수 있음

​

피자 업체 X 는 크기가 늘어날 때,

모든 비율을 똑같이 반영하고

​

Y 업체는 크기에 따라, 재료를

줄인다는 **도메인 지식** 이 있다면

​

계산에 있어서 그걸 반영할 수 있고,

만약 내가 **'생산자'** 입장에 놓인다면

​

크기가 늘어날 때마다,

​

얼마나 보간할 것인지를

계산해서 생산할 수 있음

​

**크다고 무조건 이득이 아니란 것**

**​**

**\* 본인 머리로 생각했으면 당연한 일**

**​**

4. 피자 넓이와 **'금반지'** 는?

철로 만든 원형 구체는?

**​**

**'계산하는'** 맥락에서 차이가 없음

​

**\* 내가 산 아파트의 실측 넓이가,**

**진짜 업체에서 제공한 것과 같은가?**

**같은 문제는 원을 몰라도 할 수 있음**

**​**

만약 지름이 늘어나면

무조건 더 이익이다,

​

정도로 생각한 사람이라면

​

금반지 둘레에 맞는 것도

틀리게 적용할 수 있고,

​

이래서 **'본인이'** 알아야 됨
